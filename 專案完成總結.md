## 🚀 使用方式

### 本地開發

1. **啟動應用**
   ```cmd
   雙擊運行：start-dev.bat
   ```

2. **訪問應用**
   - 首頁：http://localhost:8080
   - API 文檔：http://localhost:8080/swagger-ui/index.html

3. **測試帳號**
   - 管理員：`admin` / `admin123`
   - 業主：`owner1` / `owner123`
   - 用戶：`user1` / `user123`

### Zeabur 部署

1. **修改代碼**
   - 在本地開發並測試

2. **推送到 GitHub**
   ```bash
   git add .
   git commit -m "你的修改說明"
   git push
   ```

3. **自動部署**
   - Zeabur 會自動檢測推送
   - 自動重新部署
   - 無需手動操作

4. **訪問應用**
   - 訪問你的 Zeabur 網址

### Zeabur 環境變數（已設定）

| 變數名稱 | 說明 |
|---------|------|
| `DATABASE_URL` | MySQL 連接 URL |
| `DB_USERNAME` | MySQL 用戶名 |
| `DB_PASSWORD` | MySQL 密碼 |
| `MAIL_HOST` | Gmail SMTP 主機 |
| `MAIL_PORT` | SMTP 端口 |
| `MAIL_USERNAME` | Gmail 帳號 |
| `MAIL_PASSWORD` | Gmail 應用程式密碼 |
| `MAIL_FROM` | 寄件者郵箱 |
| `MAIL_FROM_NAME` | 寄件者名稱 |
| `APP_BASE_URL` | 應用程式網址 |
| `DDL_AUTO` | Hibernate DDL 模式 |
| `SHOW_SQL` | 是否顯示 SQL |
| `PORT` | 服務端口 |
